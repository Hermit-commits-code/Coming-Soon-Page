# Frontend Mentor - Base Apparel Coming Soon Page

This is my solution to the Base Apparel coming soon page challenge on Frontend Mentor.

## Overview

The goal was to match the provided desktop and mobile designs as closely as possible and add form validation behavior.

Users can:

- View an optimized layout for desktop and mobile screen sizes
- See styled interactive elements (input and submit button)
- Submit the email form and get validation feedback when the field is empty
- Submit the email form and get validation feedback when the email format is invalid

## Built With

- Semantic HTML5
- CSS3
- Flexbox
- CSS custom properties
- Responsive images with `picture`
- Vanilla JavaScript (form validation)

## What I Learned

- Building responsive two-column layouts that convert cleanly to a stacked mobile layout
- Keeping layout and spacing consistent using reusable CSS variables
- Structuring a form for accessibility (`aria-live`, `aria-invalid`, `aria-describedby`)
- Handling form submission with JavaScript and validating user input step by step

## Challenges I Ran Into

- Getting the desktop proportions and spacing to feel close to the design
- Positioning the submit button so it sat correctly with the input across screen sizes
- Making sure the right image source was used for desktop vs mobile
- Wiring form validation so empty and invalid states showed clear feedback

## Reflection

This challenge helped me connect styling and behavior in one project. The CSS side taught me a lot about visual precision, and the JavaScript side helped me understand practical form validation flow.
